自动从 NGA 下载回帖里的所有图片，让你从此不再错过女菩萨

## Installation

### Prerequisites
python3.8 or later

httpx>=0.23

```bash
pip install httpx==0.23
```
## Quickstart

使用命令行

```bash
cd buddhisattva
python main.py [tid]
```

例如

```bash
cd buddhisattva
python main.py 35383082
```

或者直接修改main 文件夹内的tid

图片文件会下载到content文件夹下
